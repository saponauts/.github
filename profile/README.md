# 🚀 saponauts — AI Agents für SAP-Entwicklung

Plugins und MCP-Server, die SAP ABAP-Entwicklung mit AI Agents beschleunigen. Kompatibel mit **Claude Code**, **GitHub Copilot**, **Codex**, **Cursor** und 30+ weiteren Plattformen.

## Quick Start

```
/plugin marketplace add saponauts/marketplace
/plugin install sap-abap-developer@saponauts
```

## Repos

| Repo | Beschreibung |
|------|-------------|
| [marketplace](https://github.com/saponauts/marketplace) | Plugin-Katalog — einmal hinzufügen, alle Plugins installieren |
| [sap-abap-developer](https://github.com/saponauts/sap-abap-developer) | Agents, Skills und Commands für SAP ABAP-Entwicklung |
| [mcp-abap-adt](https://github.com/saponauts/mcp-abap-adt) | MCP Server für SAP ADT — liest, schreibt und aktiviert ABAP-Objekte |

## Skills sind Cross-Platform

Die Skills in unseren Plugins basieren auf dem offenen [Agent Skills Standard](https://agentskills.io). Sie funktionieren nicht nur in Claude Code, sondern auch in **GitHub Copilot**, **OpenAI Codex**, **Cursor**, **Gemini CLI** und 30+ weiteren AI-Plattformen, die Markdown-basierte Skills unterstützen.
